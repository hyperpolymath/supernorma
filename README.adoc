== Overview

*Supranorma* is a ReScript + Deno codebase containing multiple integrated modules designed to supercharge your development workflow:

- *AI Core* - AI integration library for code analysis, generation, and intelligent tooling
- *CLI* - Powerful command-line interface with AI-powered features
- *Data Framework* - High-performance data processing and transformation framework
- *Web App* - Full-stack application for project management and collaboration
- *Dev Tools* - Developer productivity suite for automation and code quality

== Modules

=== Shared
Core utilities used across all modules:
- Logger with structured logging
- Error handling with custom types
- Async utilities (Queue, Semaphore, Mutex)
- Validators and configuration management

=== AiCore
AI integration library providing:
- LLM integration (Anthropic Claude, OpenAI GPT)
- Code analysis and understanding
- AI-powered code generation
- Intelligent refactoring suggestions
- Prompt templates and chat management

=== Cli
Command-line interface featuring:
- `analyze <file>` - AI-powered code analysis
- `generate` - Generate code from description
- `chat` - Interactive AI chat
- `explain <file>` - Explain what code does
- `refactor <file>` - Get refactoring suggestions
- `test <file>` - Generate tests for code
- `init` - Initialize configuration

=== DataFramework
Data processing framework with:
- Composable ETL pipelines
- Multiple sources (JSON, CSV, JSONL, Array, Generator)
- Multiple sinks (JSON, CSV, JSONL, Console, Array, Callback)
- Transformers (map, filter, rename, pick, omit, etc.)
- Validators and aggregators

=== WebApp
Full-stack web application:
- Oak backend with JWT authentication
- ReScript React frontend
- Project and task management
- AI assistant integration

=== DevTools
Developer productivity suite:
- Git automation with conventional commits
- Project scaffolding from templates
- Code quality checking

== Architecture

This is a ReScript codebase running on Deno. Each module is independently usable but they work together seamlessly.

```
supranorma/
├── src/
│   ├── shared/           # Core utilities
│   ├── ai-core/          # AI integration
│   ├── cli/              # Command-line interface
│   ├── data-framework/   # Data processing
│   ├── dev-tools/        # Developer tools
│   ├── web-app/          # Full-stack app
│   │   ├── backend/      #   Oak server
│   │   └── frontend/     #   React app
│   └── mod.res           # Main entry point
├── docs/                 # Documentation
├── .github/workflows/    # CI/CD
├── deno.json            # Deno configuration
├── rescript.json        # ReScript configuration
└── CLAUDE.md            # AI assistant guide
```

== Language Policy

This project enforces strict language policies:

- *ReScript only* - No TypeScript files allowed
- *Deno only* - No npm/Node.js dependencies
- *No Python* - Python files are forbidden

CI/CD workflows automatically block any commits containing forbidden file types.
